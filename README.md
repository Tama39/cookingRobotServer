# cookingRobotServer
## introduction
This server control robot arm with http request

## requirements
 - make
 - python (>= 3.7)
 - pipenv
## installing
 ```
 pipenv install
 ```
## execute server
 ```
 make run_server
 ```

## execute robotController
### executing server on heroku
 ```
 make run_robot
 ```
### executing server on local
```
make trun_robot
```
## execute test
 ```
 make test
 ```
